Typography Overview

I have chosen 3 primary fonts to use for this website. The webpage name will be a bold font at the top of the site so it stands out and can be utilitzed in conjunction with the logo for branding purposes.
The font used for the paragraphs, navigation bar and the footer of the website will be a condensed, sans-serif type. It works well in the navigation bar as well as in the body of website. This font is rounded on the ends which is a contrast to the straight edges of the website header font.
The least used font will be for the column headers. It will help to build separation of content between the navigation bar and the rest of the text below the header. It is a more traditional serif font style so it is completely different from the header and the rest of the site.
